## 📦 Description

**Auto Downloader** is a client-side utility mod that simplifies setup for modpacks, servers, or custom experiences by automatically downloading required files when they are missing.

Instead of manually installing content, players are presented with a clear and user-friendly prompt to download everything they need.

---

## ✨ Features

- 🚀 **Smart download prompt**  
  Displays a clean UI asking users to download required content when needed.

- 📥 **Automatic downloads**  
  Supports downloading:
  - Mods  
  - Resource Packs  
  - Shader Packs  
  - Save files  

- ⚙️ **Config-driven system**  
  Easily define what files should be downloaded.

- 🧠 **Smart management**  
  Handles downloads and organizes files into the correct folders.

- 🖥️ **Client-side only**  
  No server installation required.

---

## 🛠️ How It Works

1. Launch the game with the mod installed  
2. Missing files are detected  
3. A download prompt appears  
4. Files are automatically installed in the correct locations
